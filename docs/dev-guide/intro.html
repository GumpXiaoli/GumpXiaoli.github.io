What is Athene Library​

Shortly, Athene is one LBM solver framework library using block-structured AMR technique written mainly in C++.

Setup Development Environment​

信息
  • Use git as the source control tool.
  • Use VS Code as the main editor. [Recommend]
  • Use CMake as the cross-platform build chain-tool.
  • Use GoogleTest as the unit-test tool.
  • Use clang-format as the format tool.
  • Use clang-tidy as the linter tool.

A Quick Guide to Athene​

Prerequisites​

信息
  • Environment Modules is strongly recommended to provide the user environment modification. [Linux platform].
  • Python is needed. (Python3 is better.)
  • C++17 standard (or above) is enforced. Common compliers include: gcc (mingw on Windows), icc, clang, etc. Please check whether your compiler supports c++17.
  • MPI library (at least MPI2 standard) is used. Please use modules to manage different implementation.
  • CUDA computation is supported now.

Build and Test​

Configure and build are extremely easy by using cmake.

mkdir build
cd build
cmake -S ../ -B ./ -G "Unix Makefiles"
cmake --build ./ -j

Most build options are in the path Tools/CMake/AtheneOptions.cmake.

Sources Formatter and Linter​

clang-format and clang-tidy are among the most useful tools for c++ developers. They are in the llvm tools package.

Format configurations are in the file .clang-format under the root path.

.clang-format
# The style used for all options not specifically set in the configuration.
# LLVM, Google, Chromium, Mozilla, WebKit, Microsoft
BasedOnStyle: Google

ColumnLimit: 120
IndentWidth: 4
SpaceBeforeCpp11BracedList: true

Lint options are in the file .clang-tidy under the root path, and we can run the script file check_clang_tidy.sh for checking.

.clang-tidy
HeaderFilterRegex: SUP_|Work|Tests
Checks: -*,
  bugprone-undefined-memory-manipulation,
  clang-analyzer-*,-clang-analyzer-optin.mpi.*,-clang-analyzer-security.insecureAPI.strcpy,
  readability-identifier-naming,modernize-use-override,modernize-use-nullptr,
  performance-inefficient-string-concatenation
WarningsAsErrors: readability-identifier-naming
CheckOptions:
  - { key: readability-identifier-naming.NamespaceCase, value: lower_case }
  - { key: readability-identifier-naming.ClassCase, value: CamelCase }
  - { key: readability-identifier-naming.StructCase, value: CamelCase }
